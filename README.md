# Rocketshoes - First Challenge of Chapter Two

## About challenge 🤷
The main goal in this challenge is to create a store shopping cart hook to implement all the features needed for a product choice experience. The features requested are:

- Add a new product
- Remove a product
- Change the quantity of a product
- Calculation of sub-total and total cart prices
- Inventory Validation
- Displaying error messages

## Fake API with JSON Server
[JSON Server](https://github.com/typicode/json-server) is a library capable of creating a Fake API in 30 seconds and without having to write any lines of code. In this application JSON Server will simulate an API that has the information of genres and movies.

## LocalStorage API
[LocalStorage](https://developer.mozilla.org/pt-BR/docs/Web/API/Window/localStorage) is an API that allows us to persist data in the browser in a key-value scheme, similar to a JSON object. This data does not expire, that is, even if we close the application the data is not deleted.

## Toastify
[Toastify](https://fkhadra.github.io/react-toastify/introduction/) is a react package, [***react-toastify***](https://github.com/fkhadra/react-toastify#readme), which implements simple and beautiful design alerts, made to pass a direct warning message, which do not take up screen space and draw customer attention.

## 🖥️ Technologies
This project was developed with the following technologies:

- TypeScript
- ReactJS
- Styled Components
- JSON Server
- Toastify
- LocalStorage API

### 🚀 Getting Started with JSON Server

Run on the terminal to install the dependencies:
```bash
$ yarn
```

Run on the terminal to start JSON Server:
```bash
$ yarn server
```
Then you will see the message:
